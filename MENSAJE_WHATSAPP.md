## 🛡️ SCRIPTS para posibles preguntas/respuestas de la clienta

### Si pregunta: *"¿Cuánto cuesta cada opción?"*

```
Sra. Karina, justamente eso depende de la combinación
que elija. Por eso prefiero confirmar con usted los 3
puntos primero y luego enviarle la propuesta con el
precio exacto.

Para que tenga una referencia:
- El sistema básico (manual + escritorio + 1 PC) está
  en S/. 5,500
- El más completo (automático + escritorio y web + 3 PCs)
  puede llegar a S/. 9,500

Sin embargo, no es de una a la otra: cada decisión
suma o resta un monto específico que le detallo en la
propuesta formal.
```

### Si pregunta: *"¿Qué me recomiendas tú?"*

```
Sra., en base a lo que me comentó (una bodega con
fotocopiadora):

✅ Pagos digitales MANUAL para empezar (gratis, y migrar
   a automático más adelante si crece el volumen)
✅ Aplicación de ESCRITORIO Windows (más rápido para el
   POS, sin depender de internet)
✅ 2 PCs en red local (una para la caja del vendedor y
   otra para usted como administrador)

Con esa combinación el sistema queda profesional,
funcional y a un precio razonable.
```

### Si pregunta: *"¿Yape automático vale la pena?"*

```
Honestamente, Sra., al principio NO. Le explico:

- El automático cobra ~2% por cada venta Yape
- Si vende S/. 5,000 al mes en Yape, son S/. 100/mes
  en comisiones (S/. 1,200/año)
- El manual es gratis y funciona perfecto: el vendedor
  ve la notificación en su celular y escribe el código

Mi sugerencia: empezamos con manual. Cuando el negocio
crezca (más de S/. 15,000/mes en Yape), evaluamos
cambiar al automático. La integración la implemento
después como una actualización menor.
```

### Si pregunta: *"¿Por qué la web encarece más?"*

```
La versión web requiere desarrollar una segunda
interfaz adaptada a navegador (Chrome, Edge), hospedar
el servidor en la nube y mantener la sincronización
con la versión de escritorio.

Es como tener 2 productos en lugar de 1.

Si su prioridad es trabajar en la tienda, le recomiendo
solo escritorio. Si necesita ver reportes desde su casa
o celular, ahí sí justifica la inversión adicional.
```

### Si pregunta: *"¿Cuánto demora la entrega?"*

```
Depende de la complejidad final, Sra., pero en líneas
generales:

- Sistema básico (escritorio + manual + 1 PC):
  3 a 4 semanas

- Sistema completo (web + escritorio + automático +
  multi-sucursal):
  6 a 8 semanas

El detalle exacto va en la propuesta una vez definamos
el alcance.
```
